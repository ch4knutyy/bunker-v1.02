# 🏚️ BUNKER ONLINE

Онлайн-гра в стилі "Бункер", де гравці отримують унікальні характеристики та вирішують, хто виживе.

---

## 🚀 Функціонал

* Генерація персонажів (вік, професія, здоров'я, хобі, секрети)
* Реальний час через SignalR
* Система розкриття характеристик
* Карти (заміна, крадіжка, зміна характеристик)
* Голосування та виключення гравців

---

## 🛠️ Технології

* ASP.NET Core
* SignalR (real-time)
* HTML / CSS / JavaScript
* JSON (для даних гри)

---

## 📂 Структура проекту

```
bunker-project/
 ├── Controllers/
 ├── Models/
 ├── Views/
 ├── wwwroot/
 │   └── data/
 ├── Service/
 └── Program.cs
```

---

## ▶️ Як запустити

1. Відкрити проект у Visual Studio
2. Запустити (`F5`)
3. Відкрити браузер → http://localhost

---

## 📦 Дані гри

Всі характеристики зберігаються в:

```
wwwroot/data/
```
